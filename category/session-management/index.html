Python API for Sessions

📄️ create_session

package: ndb.sessionclient

📄️ end_session

Ends a session.

📄️ end_all_sessions

Ends all sessions.

📄️ session_exists

Given an array of tokens, returns those that exist.

📄️ session_info

Returns information for a session.

📄️ session_info_all

Returns minimal information for all sessions.

📄️ session_save

Save one or multiple sessions.

📄️ session_load

Restore sessions that were saved with session_save().
